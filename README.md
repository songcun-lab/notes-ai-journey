# notes-ai-journey

> 一个中年非科班者转向 AI 的**学习随笔**：按时间记录真实的认知变化与心得。

这是 [Songcun Lab](https://github.com/songcun-lab) 下偏「叙事 / 复盘」的笔记仓库，区别于工具与提示词类仓库：这里写**过程与感受**，而非可复用方法。

## 📁 建议结构

```
notes-ai-journey/
├── README.md
└── journal/
    └── 2026-06.md   # 按月归档的学习随笔
```

## ✍️ 写法

- 按月一个文件，记录这段时间学了什么、卡在哪、想通了什么；
- 重在真实，不必工整；
- 有沉淀价值的方法，再迁移到 `codex-notes` 或 `lab-ai-tools`。

> 💡 **整理建议**：本仓库目前内容很少。如果不想单独维护，可考虑把内容并入 [ai-skill-index](https://github.com/songcun-lab/ai-skill-index) 的「学习日志」板块，然后归档本仓库，减少主页上的空壳仓库。
